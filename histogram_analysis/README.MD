# Histogram İşlemleri

Bu proje, gri tonlamalı bir görüntünün histogramını hesaplayıp histogram eşitleme yöntemiyle kontrastını artırmayı amaçlar.  
Eşitleme öncesi ve sonrası görüntüler ile histogram farkı görsel olarak karşılaştırılmıştır.

## Amaç
Düşük kontrastlı görüntülerde parlaklık dağılımını iyileştirip detayları daha görünür hale getirmek.

## Kullanılan Kütüphaneler
- OpenCV  
- NumPy  
- Matplotlib
